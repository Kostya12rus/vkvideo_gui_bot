# VKVideo GUI Bot

Приложение на Python для авто просмотра стримеров VKVideo.

## Установка
1) Скачать скрипты автообновления и запуска программы [vkvideo_gui_bot.rar](https://github.com/Kostya12rus/vkvideo_gui_bot/releases/download/1.0.0/vkvideo_gui_bot.rar)
или [vkvideo_gui_bot.zip](https://github.com/Kostya12rus/vkvideo_gui_bot/releases/download/1.0.0/vkvideo_gui_bot.zip)
2) Разархивировать папку `vkvideo_gui_bot` в удобное место
3) Установка через скрипт [update_install.bat](update_install.bat) или команда `pip install --force-reinstall git+https://github.com/Kostya12rus/vkvideo_gui_bot.git`

## Использование
### GUI приложение
Запуск через скрипт [start_gui.bat](start_gui.bat) или команду `vkvideo_gui_bot_gui`
### Консольная версия
Запуск через скрипт [start.bat](start.bat) или команду `vkvideo_gui_bot_console`
